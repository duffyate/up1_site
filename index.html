1.1 Задача линейного алгоритма

Условие: Задумали число. Если его увеличить на 15, а результат умножить на 4, получится 100. Какое число задумали?

Технологии: Java
public class Task1_1 { public static int findNumber() throws ArithmeticException { int result = 100; if (4 == 0) { throw new ArithmeticException("Ошибка: деление на ноль."); } int x = result / 4 - 15; return x; } public static void main(String[] args) { try { int number = findNumber(); System.out.println("Задуманное число: " + number); } catch (ArithmeticException e) { System.out.println("Ошибка вычисления: " + e.getMessage()); } } }

1.2 Задача разветвляющегося алгоритма

Условие: Минимум из двух чисел. Сравните два числа и выведите меньшее.

Технологии: Java
import java.util.Scanner; import java.io.IOException; public class Task1_2 { public static int findMin(int a, int b) throws IOException { if (a == b) { throw new IOException("Оба числа равны — минимального нет."); } return (a < b) ? a : b; } public static void main(String[] args) { Scanner scanner = new Scanner(System.in); try { System.out.print("Введите первое число: "); int a = scanner.nextInt(); System.out.print("Введите второе число: "); int b = scanner.nextInt(); int min = findMin(a, b); System.out.println("Минимум: " + min); } catch (IOException e) { System.out.println("Ошибка сравнения: " + e.getMessage()); } catch (Exception e) { System.out.println("Ошибка ввода: введено не число."); } finally { scanner.close(); } } }

1.3 Задача циклического алгоритма

Условие: Таблица умножения. Выведите таблицу умножения для числа K (от 1 до 10).

Технологии: Java
import java.util.Scanner; public class Task1_3 { public static void printTable(int k) throws IllegalArgumentException, OutOfMemoryError { if (k <= 0) { throw new IllegalArgumentException("Число должно быть положительным."); } if (k > 10000) { throw new OutOfMemoryError("Слишком большое число: может закончиться память."); } for (int i = 1; i <= 10; i++) { System.out.println(k + " * " + i + " = " + (k * i)); } } public static void main(String[] args) { Scanner scanner = new Scanner(System.in); try { System.out.print("Введите число K: "); int k = scanner.nextInt(); printTable(k); } catch (IllegalArgumentException e) { System.out.println("Ошибка: " + e.getMessage()); } catch (OutOfMemoryError e) { System.out.println("Ошибка памяти: " + e.getMessage()); } catch (Exception e) { System.out.println("Ошибка ввода: введите целое число."); } finally { scanner.close(); } } }

1.4 Задача на проектирование классов

Условие: Класс «Студент». Создайте класс Student с полями name, age, grade с методом для вывода информации о студенте.

Технологии: Java
Student.java public class Student { private String name; private int age; private double grade; public Student(String name, int age, double grade) throws InvalidFieldException { if (name == null || name.isEmpty()) { throw new InvalidFieldException("Имя не может быть пустым."); } if (age <= 0) { throw new InvalidFieldException("Возраст должен быть положительным."); } if (grade < 0 || grade > 5) { throw new InvalidFieldException("Оценка должна быть в диапазоне 0–5."); } this.name = name; this.age = age; this.grade = grade; } public void printInfo() { System.out.println("Имя: " + name); System.out.println("Возраст: " + age); System.out.println("Оценка: " + grade); } } InvalidFieldException.java public class InvalidFieldException extends Exception { public InvalidFieldException(String message) { super(message); } } StudentDemo.java import java.util.Scanner; public class StudentDemo { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); try { // Ввод данных студента System.out.print("Введите имя студента: "); String name = scanner.nextLine(); System.out.print("Введите возраст: "); int age = scanner.nextInt(); System.out.print("Введите оценку (0–5): "); double grade = scanner.nextDouble(); Student student = new Student(name, age, grade); student.printInfo(); } catch (InvalidFieldException e) { System.out.println("Ошибка данных: " + e.getMessage()); } catch (Exception e) { System.out.println("Ошибка ввода: неверный формат."); } finally { scanner.close(); } } }

1.6 Ассемблер

Условие: Задача 1. N = N * 2. Задача 2. x:= x + a

Технологии: Assembly (NASM 2.16)
-------------------- Задача 1. N = N * 2 -------------------- bits 64 section .data N dq 15 msg db "N * 2 = " len equ $ - msg section .text global _start _start: ; Основное решение: N := N * 2 mov rax, [N] shl rax, 1 mov [N], rax ; Вывод сообщения mov rax, 1 mov rdi, 1 mov rsi, msg mov rdx, len syscall ; Конвертация числа в строку mov rax, [N] mov rcx, 10 mov rsi, rsp sub rsp, 20 xor rbx, rbx convert: xor edx, edx div rcx add dl, '0' push rdx inc rbx cmp rax, 0 jne convert output: pop rax mov [rsi], al mov rax, 1 mov rdi, 1 mov rdx, 1 syscall dec rbx jne output mov rax, 60 xor rdi, rdi syscall -------------------- Задача 2. x := x + a -------------------- bits 64 section .data x dq 20 a dq 15 msg db "x = " len equ $ - msg msg2 db ", a = " len2 equ $ - msg2 msg3 db ", x+a = " len3 equ $ - msg3 newline db 10 section .text global _start ; Процедура Padd(x, a) Padd: mov rax, [rdi] ; rax = *x add rax, rsi ; rax = *x + a mov [rdi], rax ; *x = rax ret _start: ; Вывод "x = " mov rax, 1 mov rdi, 1 mov rsi, msg mov rdx, len syscall ; Вывод x mov rax, [x] call print_num ; Вывод ", a = " mov rax, 1 mov rdi, 1 mov rsi, msg2 mov rdx, len2 syscall ; Вывод a mov rax, [a] call print_num ; Вызов процедуры Padd mov rdi, x mov rsi, [a] call Padd ; Вывод ", x+a = " mov rax, 1 mov rdi, 1 mov rsi, msg3 mov rdx, len3 syscall ; Вывод результата mov rax, [x] call print_num ; Вывод новой строки mov rax, 1 mov rdi, 1 mov rsi, newline mov rdx, 1 syscall ; Выход mov rax, 60 xor rdi, rdi syscall ; Вывод числа из rax print_num: push rbx push rcx push rdx push rsi push rdi sub rsp, 24 mov rbx, rsp mov rcx, 10 xor r8, r8 ; счётчик цифр convert_loop: xor edx, edx div rcx add dl, '0' mov [rbx + r8], dl inc r8 test rax, rax jnz convert_loop ; Выводим в обратном порядке dec r8 output_loop: movzx eax, byte [rbx + r8] mov [rsp + 20], al mov rax, 1 mov rdi, 1 lea rsi, [rsp + 20] mov rdx, 1 syscall cmp r8, 0 je done dec r8 jmp output_loop done: add rsp, 24 pop rdi pop rsi pop rdx pop rcx pop rbx ret
